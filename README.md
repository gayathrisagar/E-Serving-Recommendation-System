# E-Serving-Recommendation-System
E-Serving College Recommendation System
Tech Stack
Python, HTML/CSS (basic UI planned), SQLite (planned for DB integration)
Description
A simple recommendation system based on student academic scores to suggest suitable colleges.
Features
- Takes student marks as input
- Recommends colleges based on score range
- Can be expanded to include ranking & placements dataset
Future Enhancements
- Frontend UI using HTML/CSS
- College database integration using SQLite
